Requirements
------------

Hard requirements
^^^^^^^^^^^^^^^^^

DeepChem officially supports Python 3.8 through 3.10 and requires these packages on any condition.

- `joblib`_
- `NumPy`_
- `pandas`_
- `scikit-learn`_
- `SymPy`_
- `SciPy`_
